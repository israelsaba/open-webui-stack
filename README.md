## Unofficial Open WebUI + Anthropic (OpenSDK) Docker Compose

This repository provides a Docker Compose setup that runs **Open WebUI** alongside an **OpenSDK server** for connecting Open WebUI to **Anthropic (Claude)** via an OpenAI-compatible interface.

- Focus: simple local/self-hosted deployment
- Includes: `docker-compose.yml` and related configuration
- Not included: any claim of affiliation or endorsement
